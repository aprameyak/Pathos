## 🔧 **Technical Differences**

### V1 Architecture
```
Chrome Extension → Backend Server → DeepFace → Response
     ↓                ↓              ↓         ↓
   Screen Capture → Flask API → Neural Networks → JSON
```

### V2 Architecture
```
Chrome Extension → TensorFlow.js → Response
     ↓                ↓           ↓
   Screen Capture → face-api.js → Canvas Overlay
```
