Expected Behavior

  • Emotion boxes should appear around detected faces
  • Each box should be color-coded by emotion
  • Confidence scores should be displayed
  • Status messages should appear during analysis

Troubleshooting

  • If no boxes appear, check if the extension is running
  • If backend errors occur, wait a few seconds and try again
  • Make sure you're on a regular webpage (not chrome:// pages)
  • Check the browser console for any error messages
